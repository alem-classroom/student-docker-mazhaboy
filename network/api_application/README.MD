# Go API: book store

### Requirements:
- go

### Initiate dependencies:
```bash
go mod init
```

### Run application:
To run the application you MUST specify following env variables
```bash
PORT
DB_USER
DB_PASSWORD
DB_HOST
DB_PORT
DB_DATABASE
```

Build _go_ application
```bash
go build
```

Start application
```bash
./api_application
```
